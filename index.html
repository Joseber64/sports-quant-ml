DeepGitHub

Repositorio de Análisis Cuantitativo y Machine Learning Deportivo

Enfoque del Sistema

Este entorno aloja la infraestructura de extracción de datos, optimización de modelos predictivos y ejecución de bots automatizados para los principales mercados deportivos.

Ligas y Datasets

  • NBA - Estadísticas avanzadas y proyecciones de totales.
  • MLB - Análisis sabermétrico.
  • Liga MX - Modelado de Expected Goals (xG).

Arquitectura y Modelos

  • Optimización de cuotas mediante el Criterio de Kelly.
  • Proyección de marcadores utilizando la Distribución de Poisson.
  • Pipelines automatizados vía GitHub Actions.
